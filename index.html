Hosting the website with apache2

Installing and setting up apache2 in Linux based Machine

Apache HTTP Server is a free and open-source web server that delivers web content through the web page. A web server is software and hardware that uses HTTP (Hypertext Transfer Protocol) and other protocols to respond to client requests made over the World Wide Web. Commands to install Apache2 server in ubuntu

1. sudo apt install apache2

2. sudo systemctl start apache2

Copying website file in apache2 default webroot directory

1. cp Web_file /var/www/html

Changing Default Directory Root of apache2 to your choice of directory

1. Update the directory root of apache2.conf -> sudo nano /etc/apache2/apache2.conf eg.
2. Update the directory root of sites-available file
